API Basejump: Timestamp microservice

User stories:
    1) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
    2) If it does, it returns both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://basejump-vg.herokuapp.com/api/unixtime/December%2015,%202015
https://basejump-vg.herokuapp.com/api/unixtime/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }

Basejump: Request Header Parser Microservice

User stories:
    1) I can get the IP address, language and operating system for my browser.

Example usage:

https://basejump-vg.herokuapp.com/api/whoami

Example output:

{ "ipaddress":"109.103.234.87", "language":["en-US"], "software":"Windows NT 10.0; WOW64" }
